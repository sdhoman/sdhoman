### 🐱 Featured project — ShopKitty

> *"Should I buy this?"* — ask your friends and actually get an answer.

ShopKitty is a cross-platform social shopping app. You drop in a product link, friends vote on whether you should buy (or wear) it, and you get honest opinions with comments.

| Platform | Stack |
|---|---|
| iOS | SwiftUI · async/await · UserDefaults |
| Android | Kotlin · Jetpack · Coroutines · Coil |
| Web | Vanilla JS · Express · PostgreSQL · Heroku |

**Features:**
- Single-item votes (👍 buy it / 👎 skip it)
- **Multi-item**: pit up to 5 products against each other — *Pick One* or *Rate Each*
- Context-aware vocabulary (buy vs. wear)
- OG tag scraping for instant product previews
- Shareable vote links · name persistence · offline-first local storage
- Fastlane + TestFlight for iOS delivery
